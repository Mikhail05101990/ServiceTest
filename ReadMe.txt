1 Данный проект разработан для демонстрации навыков и умений.
  Не преднаначен и не разрабатывался для запуска и соответсвия требованиям 
  окружения выдавшей задание компании.
  Программа умышленно разработана строго в соответствии и для соответсвия 
  формальному определению в техническом задании и только.
2 Для корректной работы программы небходим доступ к интернету на машине, 
  на которой будет производиться запуск.
3 Опреционная система должна поддерживать .net 8-й версии, sdk пакет должен быть установлен.
4 Для работы предполагается использовать облачный RabbitMQ, необходимо его настроить. 
  Пример настройки имеется в статье: https://fullstackcode.dev/2022/09/30/using-cloudamqp-rabbitmq-server-with-spring-boot-application/
  Ссылка на облачный сервис: https://www.cloudamqp.com/
  При создании экзмепляра сервера на AMPQ выбирайте durable, остальные параметы по умолчанию.
5 Запуск сервисов выполняется командой dotnet run из корневой папки сервиса.
6 Сервисы тестировались на Ubuntu 20.04 LTS, .net 8.0.101. 
7 Параметры подключения сервисов задаются в appsettings.json, по одному файлу конфигурации в каждой папке сервиса.
